Report Generating Manual

This module will be used by a system administrator to get the list of all patients with a specified diagnosis for a given time period.

  1. Select “Generate Report” in home menu
  2. Enter the desired diagnosis
  3. Enter the year of diagnosis
  4. Press “Generate” to generate report
