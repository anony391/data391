Login Module

This module will be used by all users to login to the system with proper privileges, and to modify their personal information and/or the password.

    Logging in:

  1. Enter your username
  2. Enter your password
  3. Press "Submit" to login

    Modify user information:

  1. Select “Manage account settings” in home menu
  2. Redirected to a form filled out with current account settings
  3. Change the desired fields
  4. Enter a new password and re-enter to confirm password
  5. Press “Submit” to modify account settings
