User Manual

Welcome to the user manual. This is a web-based data application program simulates a Radiology Information System (RIS). This application allows users with certain privileges to access and view radiology records. There are four different classes of users: patients, doctors, radiologists, and administrators. The system is divided in to 6 different modules each performing a specific task: a Login module, a User Management module, a Report Generating module, an Uploading module, a Search module, and a Data Analysis module.
